# CLI

The Rubolt CLI (``rbcli``) helps manage projects.

## Commands

- ``rbcli init <name>`` — Create a new project
- ``rbcli run <file>`` — Run a Rubolt file
- ``rbcli build`` — Build project
- ``rbcli newlib <name>`` — Scaffold a library
- ``rbcli test`` — Run tests
- ``rbcli version`` — Version info
